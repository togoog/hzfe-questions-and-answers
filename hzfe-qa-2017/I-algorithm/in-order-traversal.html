# 算法：中序遍历

中序遍历(In-Order Traversal)，也叫做中根遍历、中序周游，指先访问左（右）子树，然后访问根，最后访问右（左）子树的遍历方式。适用于树形结构，属于深度优先遍历。

例如，有如下一种树形结构：

中序遍历的序列是：a+b*c-d-e/f
